SPA "Weather forecast"

www.asuchoj.github.io/attachmentSiteWether

The Weather Forecast application is designed to provide information about weather conditions anywhere in the world. The city search is carried out by entering its name in the search field. You can also find out the weather by move the map with the cursor. Search for information occurs using two kinds of API - XML or Fetch. In addition to weather conditions, the user can see the search history and add favorite coordinates.
