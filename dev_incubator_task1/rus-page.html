Мои работы

Сайт ОАО "ВОЛАТАВТО"

www.asuchoj.github.io/volatavtoSite

Сайт разработан для ОАО "ВолатАвто". Компания занимается проектированием, разработкой конструкторской документации и изготовлением мобильных быстроразвертываемых платформ с подъемно-мачтовыми и подъемно-поворотными устройствами различного типа и назначения. Основная задача сайта - ознакомить посетителя с компанией и продукцией, предоставить контактную информацию. На разработку сайта ушло 2 недели. Информация на сайте находится на стадии проверки руководства компании, после чего сайт будет официально выложен в сеть.

"Прогноз погоды"

www.asuchoj.github.io/attachmentSiteWether

Приложение "Прогноз погоды" разработано для предоставления информации о погодных условиях в любой точке мира. Поиск города осуществляется при помощи ввода его названия в поле поиска. Также можно узнать погоду путем перемещения карты курсором. Поиск информации происходит с помощью двух видов API - XML или Fetch. Кроме погодных условий пользователь может видеть историю поиска и добавлять любимые координаты.
