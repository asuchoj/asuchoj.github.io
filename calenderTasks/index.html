Configure Calendar





Данное приложение предназначено для создания виджета календаря, который можно использовать на любой странице вставив JavaScrip код из соответсвующего блока с кодом.

Календарь можно настроить по своему пожеланию при помощи формы.

При помощи чекбокса "allow change month" можно регулировать возможность использования стрелок для листания календаря.

При помощи чекбокса "add tasks" можно регулировать возможность добавления описаний для дня в календаре.

При помощи чекбокса "remove tasks" можно регулировать возможность удаления описаний календаря.

Чекбокс "show month / year" отвечает за отображения месяца и года в календаре.

При помощи поля add year and month можно задать нужную дату в формате год, месяц который будет отображать календарь.

Последнее поле в форме даёт возможность добавления своего класса для календаря.

На странице Create отображается превью колендаря, функционал которого можно проверить на странице Calendar.
